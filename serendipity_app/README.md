# Serendipity App

Serendipity 的 Flutter 客户端。

## 技术栈

- Flutter 3.x
- Dart 3.x
- Riverpod 2.x
- Hive
- go_router

## 项目结构

```text
lib/
├── core/       # 基础设施：providers、repositories、services、utils
├── features/   # 按业务功能划分的页面与组件
├── models/     # 数据模型
└── main.dart   # 应用入口
```

## 常用命令

```bash
flutter pub get
flutter analyze
flutter test
flutter run
```

### 4. 数据一致性测试

#### 4.1 双向关联一致性

- [ ] 检查记录的 storyLineId（应该等于故事线 B 的 id）

**预期结果：**
- 旧关联正确解除
- 新关联正确建立

#### 4.3 删除操作一致性
- [ ] 删除一条关联了故事线的记录
- [ ] 检查故事线的 recordIds（应该不包含该记录）
- [ ] 删除一条包含记录的故事线
- [ ] 检查记录的 storyLineId（应该为 null）

**预期结果：**
- 删除记录时自动解除关联
- 删除故事线时自动清空所有记录的关联

5. **重启应用验证**（1分钟）

### 1. 架构违规问题（P1 - 高优先级）✅

**问题**：直接实例化 `StorageService()`，违反依赖倒置原则

**修复前**（4处）：
```dart
// 第95行：字段声明
final _storage = StorageService();

// 第379行：使用
final records = _storage.getRecordsByStoryLine(_selectedStoryLineId!);

// 第455行：使用

// 第1547行：使用
final storyLineName = _storage.getStoryLine(_selectedStoryLineId!)?.name;
```

**修复后**：
```dart
// 通过 Provider 访问数据
final recordsAsync = ref.read(recordsProvider);
final records = recordsAsync.value ?? [];

final storyLinesAsync = ref.read(storyLinesProvider);
final storyLines = storyLinesAsync.value ?? [];
```

**符合维度**：
- ✅ 1️⃣ 架构设计原则 - 依赖倒置原则
- ✅ 2️⃣ 分层约束 - UI层通过Provider访问数据

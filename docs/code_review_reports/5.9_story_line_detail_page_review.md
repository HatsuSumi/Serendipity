## ⚠️ 发现的问题

### 问题1：违反分层约束 - 直接使用底层服务（高优先级 🔥）

**违反维度**：1️⃣ 架构设计原则、2️⃣ 分层约束

**问题描述**：
UI 层直接使用 `storyLineRepositoryProvider` 和 `storageServiceProvider`，违反依赖倒置原则和分层约束。

**问题代码**（3处）：

```dart
// 第 50 行 - _loadRecords 方法
Future<void> _loadRecords() async {
  setState(() {
    _isLoading = true;
  });

  try {
    final repository = ref.read(storyLineRepositoryProvider);  // ⚠️ 直接访问 Repository
    final records = repository.getRecordsInStoryLine(_currentStoryLine.id);

    setState(() {
      _records = records;
      _isLoading = false;
    });

    setState(() {

Future<void> _refresh() async {
  final storage = ref.read(storageServiceProvider);  // ⚠️ 直接访问 StorageService
  final updatedStoryLine = storage.getStoryLine(_currentStoryLine.id);
  if (updatedStoryLine != null) {
    setState(() {
      _currentStoryLine = updatedStoryLine;
    });
  }
  await _loadRecords();
}

// 第 476 行 - _showDeleteRecordConfirmDialog 方法
onPressed: () async {
```

**问题影响**：
- 违反依赖倒置原则（应依赖Provider抽象）
- 违反分层约束（UI层直接访问数据层）
- 降低可测试性（无法mock）
- 与其他已修复页面不一致

**修复建议**：
```dart
// 使用 recordsProvider 和 storyLinesProvider
Future<void> _loadRecords() async {
  setState(() {
    _isLoading = true;
  });

  try {
    final recordsAsync = ref.read(recordsProvider);
    final allRecords = recordsAsync.value ?? [];
    
    // 筛选出属于该故事线的记录
    final records = allRecords.where((record) {
      return _currentStoryLine.recordIds.contains(record.id);
    }).toList();
    
    // 按时间排序
    records.sort((a, b) => a.timestamp.compareTo(b.timestamp));

    setState(() {
      _records = records;
      _isLoading = false;
    });
  } catch (e) {
    setState(() {
      _isLoading = false;
    });
    rethrow;
  }
}

// _refresh 方法
Future<void> _refresh() async {
  final storyLinesAsync = ref.read(storyLinesProvider);
  final allStoryLines = storyLinesAsync.value ?? [];
  final updatedStoryLine = allStoryLines.firstWhere(
    (sl) => sl.id == _currentStoryLine.id,
    orElse: () => _currentStoryLine,
  );
  
  setState(() {
    _currentStoryLine = updatedStoryLine;
  });
  
  await _loadRecords();
}

// 删除记录
await ref.read(recordsProvider.notifier).deleteRecord(record.id);
```

## ⚠️ 发现的问题

### 问题1：不必要的 toList() 调用（低优先级 💡）

**违反维度**：9️⃣ 性能检查

**问题描述**：
在 spread 操作符中使用了不必要的 `toList()` 调用，会产生额外的列表拷贝开销。

**Analyzer 警告**（2处）：
```
info - settings_page.dart:50:14 - unnecessary_to_list_in_spreads (toList)
info - settings_page.dart:89:14 - unnecessary_to_list_in_spreads (toList)
```

**问题代码**：

```dart
// 第 50 行
...PageTransitionType.values.map((type) {
  final isSelected = currentTransition == type;
  return ListTile(
    leading: Text(
      type.icon,
      style: const TextStyle(fontSize: 24),
    ),
    title: Text(type.label),
    trailing: isSelected
        ? const Icon(Icons.check, color: Colors.blue)
        : null,
    selected: isSelected,
    onTap: () {
      ref.read(pageTransitionProvider.notifier).state = type;
      MessageHelper.showSuccess(context, '已切换到：${type.label}');
    },
  );
}).toList(),  // ⚠️ 不必要的 toList()

// 第 89 行
...DialogAnimationType.values.map((type) {
  final isSelected = currentDialogAnimation == type;
  return ListTile(
    leading: Text(
      type.icon,
      style: const TextStyle(fontSize: 24),
    ),
    title: Text(type.label),
    trailing: isSelected
        ? const Icon(Icons.check, color: Colors.blue)
        : null,
    selected: isSelected,
    onTap: () {
      ref.read(dialogAnimationProvider.notifier).state = type;
      MessageHelper.showSuccess(context, '已切换到：${type.label}');
    },
  );
}).toList(),  // ⚠️ 不必要的 toList()
```

**问题影响**：
- 产生不必要的列表拷贝
- 轻微的性能开销（虽然在这个场景下影响很小）
- 代码不够简洁

**修复建议**：

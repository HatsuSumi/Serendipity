## ⚠️ 发现的问题

### 问题1：违反分层约束（高优先级 🔥）

**违反维度**：1️⃣ 架构设计原则、2️⃣ 分层约束

**问题描述**：
UI 层直接实例化并使用 `StorageService()`，违反依赖倒置原则和分层约束。

**问题代码**（4处）：

```dart
// 第95行：字段声明
final _storage = StorageService();

// 第379行：使用
final records = _storage.getRecordsByStoryLine(_selectedStoryLineId!);

// 第455行：使用
final records = _storage.getAllRecords();

// 第1547行：使用
final storyLineName = _selectedStoryLineId != null
    ? _storage.getStoryLine(_selectedStoryLineId!)?.name
    : null;
```

**问题影响**：
- 违反依赖倒置原则（应依赖Provider抽象）
- 违反分层约束（UI层直接访问数据层）
- 降低可测试性（无法mock）
- 与其他已修复页面（timeline_page、record_detail_page）不一致

**修复建议**：
```dart
// 删除字段声明
// final _storage = StorageService(); // ❌ 删除

// 通过Provider访问
final records = ref.read(recordsProvider.notifier).getRecordsByStoryLine(id);
final allRecords = ref.watch(recordsProvider).value ?? [];
final storyLine = ref.read(storyLinesProvider.notifier).getStoryLine(id);
```
